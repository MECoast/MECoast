BR Class 5MT 4-6-0 steam locomotive

3D Model built from drawings and textured by Dave Robinson
Cab view by Lew Price 
Sounds By Stuart Williamson 
Tested by Ben Harrison

-----------------------------------------------------------------------------
The 3D model has been built using TrainSim Modeller

This is a model of a BR Class 5MT 4-6-0 44806 
In BR Plain Black Livery

This loco comes with changeable headcode simple copy over all the dead lamps first from the dead lamp 
folder in to the main folder and then copy from the live lamps folder the lamps you require on the loco to make the headcode 
to the main folder.
It also has removerable nameplates for those who dont want them on which work in the sameway.
The cabview is supplied with thanks from Lew Price and you will requier Stuart Williamson's 2 Cyilnder sound from UKTS file id 2910

To install simpley copy the 3 files ( DR_BR_5MT_44806, common.sound, common.cabs) to your Trainset folder which can be found here
C:\Program Files\Microsoft Games\Train Simulator\TRAINS\Trainset 
you will then need to make a consists using activtie editor in Train Simulator Editors & Tools
