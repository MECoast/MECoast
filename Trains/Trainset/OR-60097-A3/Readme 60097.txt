Ex-LNER A3 60097 Humorist 

3D Model built from drawings and textured by Dave Robinson
Sounds By Chris Jephcott File ID:27204
Cabview By hans nooyen File ID:22338
Eng file by Rick gibbs
-----------------------------------------------------------------------------
The 3D model has been built using TrainSim Modeller

This is a model of Ex-LNER A3 60097 Humorist in BR Brunswick Green
The only A3 to be fitted with original smoke deflectors 

This loco comes with changeable headlamps, simple copy over all the dead lamps first from the dead lamp folder in to the 
main folder and then copy from the live lamps folder the lamps you require on the loco to the main folder. 

To install copy DR_BR_A3_60097 folder to your trainset which can be found here 
(C:\Program Files\Microsoft Games\Train Simulator\TRAINS\Trainset)

Any suggestions of help or constructive comments welcome.
