HISTORY
~~~~~~~
While private owners coal wagons were common before nationalisation, vans have always been rare. This wagon dates from the first world war and survived until the sixties. It was used for moving stores between depots.

These wagons require Stuart Williamson's  SWB Wagon Sounds  File ID 3290
