INSTALLATION
~~~~~~~~~~~~
Run the Installer file to install to your MSTS folder C:\Program Files\Microsoft Games\Train Simulator\Trains\Trainset.

Please note that no consists have been set up in this installation.

To create a consist that uses this model;
1) Open Activity Editor and create your consist
2) Find 'MK1_Tail_Lamp' under the 'Freight' drop down menu and add to the rear of your consist
3) Save the consists and run the activity.

The tail lamp has a self illuminated red aspect, so no need to press your 'H' button for it to work.  However if you want to get a very subtle and 
slight flicker press the 'H' Button.

Please be aware that this model may or may not be compatible with all the rolling stock that you have available. The tail lamp was primarily designed 
on the BATS MK1 Stock, which means that it may not sit correctly on other stock.
