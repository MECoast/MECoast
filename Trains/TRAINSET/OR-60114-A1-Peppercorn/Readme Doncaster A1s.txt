Pack of 4 A1s Based at Doncaster 

3D Model built from drawings and textured by Dave Robinson
Sounds By Chris Jephcott File ID:27204
Cabview By hans nooyen File ID:22338
Eng file by Rick gibbs
-----------------------------------------------------------------------------
The 3D models has been built using TrainSim Modeller

This pack contains 4 A1 which were based at Doncaster 
60114 W.P.Allen
60128 Bongrace  (Weathered)
60139 Sea Eagle (Weathered)
60158 Aberdonian

These locos comes with changeable headlamps, simple copy over all the dead lamps first from the dead lamp folder in to the 
main folder and then copy from the live lamps folder the lamps you require on the loco to the main folder. 

To install copy all the files to your trainset folder which can be found here 
(C:\Program Files\Microsoft Games\Train Simulator\TRAINS\Trainset)

Any suggestions of help or constructive comments welcome.
